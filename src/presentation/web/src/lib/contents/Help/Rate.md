# アクティブレートについて

## 数値の良し悪し

基本的に数値が高いほうがアクティブと言う事になります。

## どうやって算出しているの？

チャットやVCの活発度、メンバーのオンライン比率、新規メンバーの数などを元に独自のアルゴリズムで算出しています。

悪用される可能性があるのでで現状アルゴリズムを公開する気はありません。

## 特典

アクティブレートに応じて枠がサーバーアイコンの周りに付くようになります。
