# サーバー登録方法

## ボットをサーバーに入れる

まず以下のリンクからサーバーにボットを導入してください

[BOT_INVITE_LINK](BOT_INVITE_LINK)

ボットの権限はデフォルトで管理者になっていますがお好みで大丈夫です。

※権限によっては正確にアクティブレートを算出できない可能性があります。

## 導入後

- 1: `/web` コマンドを実行し変更から内容を公開内容を変更してください
- 2: 公開サーバーにするためには公開チェックボックスにチェックを入れる必要があります (いつでも非公開へ変更可能です。)
- 3: 保存を押すと変更が適応されます。
